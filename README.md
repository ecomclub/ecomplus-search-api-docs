# ecomplus-search-api-docs
API Blueprint to E-Com Plus Search API documentation

# Setting up
Install [Hercule](https://github.com/jamesramsay/hercule)
and give permissions to shell script:

```bash
npm install -g hercule
chmod +x build.sh
```

## Build doc files
```bash
./build.sh
```
